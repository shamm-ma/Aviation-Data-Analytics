# Flight Route & Traffic Analysis using Live ADS-B Data

## Project Overview
This project analyzes live ADS-B flight tracking data from the OpenSky Network API.

The goal is to pull real flight data, clean it, analyze regional air traffic patterns, and create an interactive flight map using Python.

## Tools Used
- Python
- pandas
- requests
- folium
- Jupyter Notebook
- VS Code

## Project Steps
1. Pull live flight data from the OpenSky API
2. Clean and filter the dataset
3. Analyze flight traffic patterns
4. Create an interactive Folium map

## Outputs
- Cleaned flight dataset
- Traffic analysis notebook
- Interactive HTML flight map

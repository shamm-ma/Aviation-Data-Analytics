# U.S. Airlines On-Time Performance Analytics

## Project Overview
This project analyzes U.S. airline on-time performance data from **April 2025 to April 2026**. The analysis covers **21 U.S. airlines** and focuses on flight delays, delay causes, airline performance, and operational trends.

## Objective
The objective of this project is to clean and analyze airline delay data, then build a Power BI dashboard that helps identify key performance issues across U.S. airlines.

## Tools Used
- Python
- pandas
- Jupyter Notebook
- Power BI
- GitHub

## Dataset
The dataset contains U.S. airline delay and on-time performance data from the Bureau of Transportation Statistics.

Time period analyzed:
- April 2025 to April 2026

Airlines analyzed:
- 21 U.S. registered airlines

## Project Workflow
1. Loaded & inspected the airline delay dataset.
2. Checked missing values, duplicate records, and column structure.
3. Cleaned & prepared the dataset for analysis.
4. Created summary datasets for airline performance, delay causes, and monthly trends.
5. Built an interactive Power BI dashboard to visualize key insights.

## Key Analysis Areas
- On-time performance by airline
- Delay rate by airline
- Delay cause breakdown
- Monthly on-time performance trends

## Files Included
- `notebooks/`  
  Contains the Jupyter Notebook used for data cleaning and analysis.
- `data/raw/`  
  Contains the original raw airline delay dataset.
- `data/cleaned/`  
  Contains cleaned and processed datasets used for analysis and dashboarding.
- `dashboard/`  
  Contains the final Power BI dashboard file.
- `references/`  
  Contains supporting project reference material.

## Dashboard
The Power BI dashboard presents airline performance trends, delay causes, and operational insights for the selected time period.

## Outcome
This project demonstrates how airline operational data can be cleaned, analyzed, and visualized to support performance monitoring and data-driven decision-making in airline operations.
